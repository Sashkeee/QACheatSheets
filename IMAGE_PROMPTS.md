# Промпты для генерации изображений

Если я не могу сгенерировать изображения из-за ограничений системы, вы можете использовать эти промпты в Midjourney, DALL-E 3 или Stable Diffusion, чтобы получить идеальные иллюстрации для сайта.

### 1. Жизненный цикл разработки ПО (SDLC)
**Промпт:**
> A modern, clean infographic diagram of the Software Development Life Cycle (SDLC) for a QA educational website. Horizontal flow: Requirements -> Design -> Development -> Testing -> Deployment -> Maintenance. Soft gradients, professional icons for each stage, Inter font. Premium look, high resolution, minimalist flat design.

**Куда вставить:** В раздел "Основы" (basics) -> "Жизненный цикл ПО".
**Имя файла:** `sdlc_diagram.png`

---

### 2. Пирамида тестирования (Testing Pyramid)
**Промпт:**
> A professional testing pyramid diagram. Bottom layer (wide): Unit Tests (fast, cheap). Middle layer: Integration/API Tests. Top layer (tip): UI/E2E Tests (slow, expensive). Modern flat design, soft colors (blue, green, orange), English labels, no people. High resolution, vector style.

**Куда вставить:** В раздел "Автоматизация" (automation) -> "Пирамида автоматизации".
**Имя файла:** `test_pyramid_diagram.png`

---

### 5. Docker и Контейнеризация
**Промпт:**
> Concept art of a blue shipping container with a whale logo, representing Docker. Inside the container are icons for Database, Code, and Web Server. Clean 3D render style, studio lighting, soft shadows, premium tech aesthetic.

**Куда вставить:** В раздел "Инфраструктура" (infrastructure) -> "Docker: Контейнеризация".
**Имя файла:** `docker_concept.png`

---

**Инструкция:**
1. Сгенерируйте картинки по этим промптам.
2. Сохраните их в папку `c:\Users\user\Documents\Antigravity\QA CheatSheet\app\public\` под указанными именами.
3. Сайт подхватит их автоматически (я уже прописал пути в коде).
