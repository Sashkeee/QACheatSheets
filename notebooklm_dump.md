# QA CheatSheet - Полная база знаний для NotebookLM
Обновлено: Март 2026

Этот документ содержит полную структуру проекта, теоретические выжимки, практические примеры и список используемой литературы проекта qacheatsheet.ru. Предназначен для глубокого анализа, генерации подкастов и подготовки к собеседованиям.

---

## 📚 Основные источники и литература
Проект базируется на проверенных временем знаниях экспертов. Ниже приведен список книг и ресурсов, использованных при создании контента:

1. **Ольга Назина** - "Что такое тестирование. Курс молодого бойца" (фундаментальная база, аналогия с кондитером, эвристики).
2. **Роман Савин** - "Тестирование dot com" (практический подход и вход в профессию).
3. **Алан Болье (Alan Beaulieu)** - "Learning SQL" (основы баз данных на примере Sakila).
4. **Ли Копланд (Lee Copeland)** - "A Practitioner's Guide to Software Test Design" (мастерство тест-дизайна).
5. **Рой Ошероув (Roy Osherove)** - "The Art of Unit Testing" (автоматизация нижнего уровня).
6. **OWASP Top 10** - Глобальные стандарты безопасности веб-приложений.
7. **Whittaker, Arbon, Carollo** - "How Google Tests Software" (инженерная культура и пирамида тестов).
8. **ISTQB Foundation Level Syllabus 4.0** - Международные стандарты и терминология.

---

## 🏗️ Проектная структура и Стек (Технический дамп)
Для понимания архитектуры:
- **Frontend**: React 19, TypeScript, Vite, Tailwind CSS.
- **Дизайн**: Glassmorphism (эффект стекла), Volumetric UI (3D-меню), Mesh Backgrounds.
- **Инфраструктура**: Docker, Caddy (авто-SSL), GitHub Actions (авто-деплой на VPS).
- **Размещение**: Host: OrangeBrew.Ru, Domain: qacheatsheet.ru.
